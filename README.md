# 🎯 Lucky Wheel 抽奖转盘系统

一个基于 HTML + CSS + JavaScript 的前端抽奖转盘网页，可用于年会、活动、课堂或任何需要随机抽取编号的场景。  
支持多奖项、动画转盘、抽奖记录，并可直接部署到 GitHub Pages 在线使用。

---

## ✨ 功能特性

- 🎡 转盘形式抽奖（1–100 编号）
- 🏆 支持多奖项  
  - 一等奖  
  - 二等奖  
  - 三等奖  
  - 四等奖
- ▶ 点击“抽奖”开始旋转
- ⏸ 点击“停下”缓慢减速并确定结果
- 📝 抽奖记录自动保存（编号 + 奖项）
- 🌐 可通过 GitHub Pages 部署为公网网页
